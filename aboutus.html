The Team

Coming together because the “Q-Man” said so, we bandied about a bunch of different ideas. Those ideas morphed into something that we thought would fill a niche in the current travel landscape - a local event finder. This required a site that utilized all the skills and technologies we had learnt in the past few months, so it was a great experience to see it all being implemented in one comprehensive project.


After a lot of camaraderie and coffee, Git push and pull, it all came together – like a Falcon Heavy blasting off into space. After we came down to Earth, we realized that we should tone down the celebrations since we still had a few more months of Bootcamp left to do.
